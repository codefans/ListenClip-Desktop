## 二、页面拓扑关系图

```
App Shell (NavigationRail + TopAppBar)
│
├── [P1] 媒体库页 Library
│     ├── → [P2] 导入向导 ImportWizard        (点击"导入"FAB)
│     ├── → [P3] AI处理页 Processing           (导入完成后自动跳转)
│     └── → [P4] 精听播放器 Player             (点击已处理的条目)
│
├── [P2] 导入向导 ImportWizard
│     ├── Step 1: 选择文件 (本地/URL)
│     ├── Step 2: 元信息填写 (标题/难度/标签)
│     └── → [P3] AI处理页                     (点击"开始处理")
│
├── [P3] AI处理页 Processing
│     ├── 转录进度显示
│     ├── 断句预览
│     ├── → [P4] 精听播放器                   (处理完成后)
│     └── → [P1] 媒体库                       (取消/后台处理)
│
├── [P4] 精听播放器 Player  ← 核心页面
│     ├── 波形 + 句子高亮轨道
│     ├── [P4-A] 当前句子卡片 (循环/跟读/注释)
│     ├── → [P5] 句子编辑器                   (长按句子 / 右键菜单)
│     └── → [P6] 导出面板                     (点击分享按钮)
│
├── [P5] 句子编辑器 SentenceEditor
│     ├── 时间轴微调 (拖拽边界)
│     ├── 字幕文本修正
│     ├── 添加词汇注释/高亮
│     └── → [P4] 返回播放器
│
├── [P6] 导出面板 ExportPanel
│     ├── 导出 SRT / LRC / Anki CSV
│     ├── 导出截选片段 (MP3/MP4)
│     └── → [P4] 返回播放器
│
└── [P7] 设置中心 Settings
      ├── AI 引擎配置 (API Key / 模型选择)
      ├── 播放偏好 (默认速度/循环次数)
      ├── 快捷键管理
      └── 存储路径管理
```

## 三、各页面功能优先级清单

### P1 · 媒体库 Library

| 优先级 | 功能 | 说明 |
|--------|------|------|
| P0 | 媒体条目列表 | 卡片式展示，含封面/时长/处理状态 |
| P0 | 点击进入播放器 | 已处理完成的条目可直接点击播放 |
| P0 | 导入 FAB 按钮 | 右下角悬浮按钮，触发导入向导 |
| P1 | 处理状态徽标 | 待处理/处理中/完成/失败 四态 |
| P1 | 搜索栏 | 按标题/标签搜索 |
| P1 | 排序与筛选 | 按日期、时长、难度排序 |
| P2 | 多选批量操作 | 批量删除/导出/重新处理 |
| P2 | 最近播放 Section | 顶部置顶最近 3 条 |
| P3 | 标签管理 | 自定义标签颜色与分类 |

## 五、关键数据流

```
[文件/URL]
    │
    ▼
ImportService
    │  音频提取 (FFmpeg)
    ▼
AudioFile (.wav / 16kHz mono)
    │
    ▼
TranscriptionService
    │  Whisper (本地 / API)
    ▼
RawTranscript { segments: [{start, end, text}] }
    │
    ▼
SentenceAlignmentService
    │  NLP 断句优化 + 词级对齐
    ▼
Project {
  media: MediaFile,
  sentences: Sentence[],   ← 核心数据模型
  annotations: Annotation[]
}
    │
    ├──▶ PlayerViewModel  (播放控制状态)
    ├──▶ EditorViewModel  (句子编辑状态)
    └──▶ ExportService    (格式转换输出)
```

## 六、技术栈建议

| 层次 | 方案 | 理由 |
|------|------|------|
| UI 框架 | **Flutter for Windows** | 原生支持 Material 3，跨平台，渲染性能优秀 |
| 音频处理 | **FFmpeg** (via dart:ffi / process) | 工业级音视频处理 |
| AI 转录 | **faster-whisper** (本地 Python 服务) 或 OpenAI Whisper API | 本地优先保护隐私 |
| 波形渲染 | **audiowaveform** 预处理 + Canvas 自绘 | 性能佳 |
| 本地数据库 | **SQLite** (drift ORM) | 轻量，无需服务端 |
| 状态管理 | **Riverpod** | 适合复杂异步状态 |
| 进程通信 | **stdio JSON-RPC** (Flutter ↔ Python) | AI 服务隔离 |
